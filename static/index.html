SQL to XORM Converter

This tool converts mysql create statements into Go type definitions.

Other SQL expressions are ignored.

Paste a create statement on the left and the equivalent Go type will be generated on the right,

which you can paste into your program. The script has to make some assumptions,

so check the output carefully!
